# OpenBlox - Open source sandbox style platform

This repository serves as the sole monorepo for the OpenBlox project, containing the OpenBlox platform (web and game clients) and all backend services.

## What is OpenBlox?

OpenBlox serves as a core foundation of a sandbox platform with features including, but not limited to:
- User accounts with avatars and inventory system
- Places (games) with versioned publishing
- Content delivery and hosting
- Managed multiplayer servers for places.

OpenBlox is built with the intention to be self-hostable and expansive with a focus on clean architecture and maintainability.

## Status

OpenBlox is in ***very* early development** and is not yet production ready.
